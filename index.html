HOPS Contour - CNC Layout Creator



This script creates a Layout for CNC laser cutting based from a geometry and board size specified (Currently only works on Grasshopper Player*doesnt work with Hops)

Parameters:
  • Geometry - Specify the Object to be contoured (Closed Solid)(MM)
  • Material Thickness - Specify the thickness of the contour material(Float)(MM)
  • Bounding Curve - Specify the Boundary to lay the curves on (Closed Curve)(MM)
  • Spacing - Specify the Spacing between geometry for Laser cut (Float)(MM)
